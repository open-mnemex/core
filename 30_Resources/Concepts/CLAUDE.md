# Concepts (概念库)

Definitions of frameworks, mental models, and terminology.

---

## Naming

`concept_name_in_snake_case.md` — one file per concept.

## Template

See `_capture_registry.md` → `concept` type for full template.

## Rules

- One concept per file
- Include origin, definition, examples, and sources
- Chinese summary if the concept was encountered in Chinese
- Link to related concepts if they exist in this folder
