# Lists (收藏清单)

Curated lists of external recommendations. One file per category.

---

## Files

| File | `/capture` Type | Description |
|------|-----------------|-------------|
| `Reading_List.md` | book | Books to read |
| `Watch_List.md` | show | Movies/TV shows |
| `Tools.md` | tool | Software and apps |
| `Music_Podcasts.md` | music | Music and podcasts |
| `Courses.md` | course | Online courses |
| `Travel_List.md` | travel | Places to visit |
| `Wishlist.md` | wishlist | Products to buy |
| `Puzzles.md` | puzzle | Logic puzzles |

---

## Rules

- **Mode:** append — always add to existing file, never overwrite
- **Checkbox:** Use `- [ ]` for actionable items (books, shows, courses)
- **No duplicates:** Search the file before appending
- **Templates:** See `_capture_registry.md` for each type's format
