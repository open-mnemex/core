# Resources (外部收藏)

External materials collected for reference. Read-only library.

> Key Question: "Is this external material I've gathered, not created?"

---

## Structure

| Folder | Type | Mode |
|--------|------|------|
| `Articles/` | Saved articles with full text | new_file |
| `Concepts/` | Concept definitions and frameworks | new_file |
| `Lists/` | Curated lists (books, shows, tools...) | append |
| `Quotes/` | Collected wisdom and quotes | append |
| `Paper_Library/` | Academic papers | new_file |
| `CS_Books/`, `Finance_Books/` etc. | Downloaded books | file |

---

## Capture Types → Destinations

All types defined in `_capture_registry.md`.
Each subfolder with capture targets has its own `CLAUDE.md`.

| `/capture` Type | Destination |
|-----------------|-------------|
| quote | `Quotes/Quotes.md` |
| book | `Lists/Reading_List.md` |
| show | `Lists/Watch_List.md` |
| tool | `Lists/Tools.md` |
| music | `Lists/Music_Podcasts.md` |
| course | `Lists/Courses.md` |
| travel | `Lists/Travel_List.md` |
| wishlist | `Lists/Wishlist.md` |
| puzzle | `Lists/Puzzles.md` |
| article | `Articles/` (new file) |
| concept | `Concepts/` (new file) |

---

## Guidelines

- Do NOT create files at the root of `30_Resources/`.
- All content goes into the appropriate subfolder.
- Follow templates in `_capture_registry.md`.
- Line width <= 80 chars.
